# PBI---ZoopMegaStore
Projeto Power BI – Análise de Dados da ZoopMegaStore

Este projeto foi desenvolvido durante um curso de Power BI pela plataforma Alura, com foco em tratamento, modelagem e análise de dados, aplicando conceitos fundamentais de BI, DAX e não focando tanto nos visuais.

Fiz o tratamento e limpeza de dados no Power Query, aplicando boas práticas para organização, padronização e preparação da base de dados. 
No modelo analítico,pratiquei o uso de DAX, com destaque para a função CALCULATE, funções agregadoras iterativas como AVERAGEX, SUMX e MAXX, além de cálculos envolvendo datas e períodos utilizando DATEDIFF, DATESYTD e DATESMTD.

Este projeto representa minha evolução prática no uso do Power BI e meu interesse em atuar com dados, análise e visualização, como tenho visto sempre nos cursos buscar sempre transformar dados brutos em informações claras e relevantes para o negócio.
